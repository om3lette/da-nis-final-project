# Web-scrapper для итогового проекта по НИСу "Анализ данных в Python"

## Как запустить

Установите зависимости из `requirements.txt`

Пример для `Linux`:

```bash
python3 -m venv .venv && \
source .venv/bin/activate && \
pip install -r requirements.txt
```

Запустите скрейпер

```bash
python3 src/main.py
```

Драйвер для selenium должен загрузится автоматически при первом запуске.

## Процесс работы

1. Загрузка дат сохранений. С помощью `Selenium` будут собраны все даты, в которые были сохранены данные сайта.
2. Сбор данных. Для каждой даты будет загружена веб страница и извлечены данные.
3. Создание `dataframe` и экспорт в `.csv` и `.xlsx`

Шаги 1 и 2 кэшируются. В автоматически созданной директории `gen/cache` хранятся обработанные с помощью
[ammonia](https://github.com/rust-ammonia/ammonia) (nh3) скачанные страницы, а также извлеченные даты сохранений.
Это позволяет второму и дальнейшим запускам выполняться без доступа к интернету.

Результирующие файлы будут сохранены в `gen/out/<website>`.

## Дополнительная информация

### Предупреждение при выполнении

Во время выполнения могут выводиться предупреждения. Это связанно с тем, что некоторые сохранения сайта сломаны, то есть
данные были утеряны или вместо ожидаемой страницы сохранена страница с ошибкой доступа.

Предупреждения не являются ошибками и их можно игнорировать.

### Конфигурация

Некоторые параметры можно задать через файл `config.yaml`. Он будет автоматически сгенерирован при первом запуске.
